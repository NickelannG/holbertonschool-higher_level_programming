The ``1-my_list`` module
======================

Using ``MyList``
-------------------
First import ``MyList`` from the ``1-my_list`` module:
    >>> MyList = __import__("1-my_list").MyList

# Check with sorted append:
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(4)
>>> my_list.append(5)
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

# Check with unsorted append:
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)

# Check with no sorted append with negative number:
>>> my_list = MyList()
